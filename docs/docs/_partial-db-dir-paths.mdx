The default storage path depends on your operation system and installation method:

- **macOS Desktop**: `/Users/<username>/.aiexec/data/database.db`
- **Windows Desktop**: `C:\Users\<name>\AppData\Roaming\com.Aiexecdata\aiexec.db`
- **OSS macOS/Windows/Linux/WSL (`uv pip install`)**: `<path_to_venv>/lib/python3.12/site-packages/aiexec/aiexec.db` (Python version may vary)
- **OSS macOS/Windows/Linux/WSL (`git clone`)**: `<path_to_clone>/src/backend/base/aiexec/aiexec.db`
