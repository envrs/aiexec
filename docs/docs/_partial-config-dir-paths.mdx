The default config directory location depends on your operating system and installation method:

- **macOS Desktop**:`/Users/<username>/Library/Logs/com.Aiexec`
- **Windows Desktop**:`C:\Users\<username>\AppData\Roaming\com.Aiexeccache`
- **OSS macOS/Windows/Linux/WSL (`uv pip install`)**:
  - **macOS**: `/Users/<username>/Library/Caches/aiexec`
  - **Linux**: `/home/<username>/.cache/aiexec`
  - **Windows**: `C:\Users\<username>\AppData\Local\aiexec\aiexec\Cache`
- **OSS macOS/Windows/Linux/WSL (`git clone`)**: `<path_to_clone>/src/backend/base/aiexec/`
